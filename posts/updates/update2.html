Update 2: New Overworld

July 4, 2017.

Welcome back everyone, this is Update 2 to Project Phases. It focuses primarily on the New Overworld. This is a much smaller update, since the biggest new change are some new graphics. So lets see what is new.

Overworld Changes

Starting up the game, you can see a big change immediately with the new overworld graphics. They are basic (and not very good looking) tile sprites. There are only two sprites, grass and dirt but it is better than what we had before (which was literally nothing).


I also added some other things to the overworld. First, signs which simply create a message when you hover over them. Second is the heart, when you touch this heart it heals you. Your health is always displayed in the top right corner, so you can check it there.


Battle Changes

There are barely any changes to the battle system besides small bug fixes. The big one is that if you don’t have enough points or time for something, a message appears at the top of the attack menu. Before it was a pop-up box, but now it is an in-game message. There is also a small temporary change for testing, which is that the enemy only has 1 health to start the battle.


Like I said, it’s a pretty small update but good none the less. The next update is Enemy AI which will help lead into Multiple Player Combat.
