Games

Project Phases (In Development)

Project Phases is my main project I've been working on for many years. It's an RPG with a one of a kind timeline based combat system. The entire game's development has been documented on this website through dev-logs and deticated update posts and videos. While it's been in development for a long time, it will be coming out soon so please stay tuned!


Parallel Portals (2022)

Parallel Portals is a game made for the Mini Jame Gam #10 based on the theme "Reality is Breaking". You are tasked with mending a broken reality by jumping between parallel universes to find 8 core crystals. Each universe has their own unique creatures and physics. You can create your own portals for reality bending results by combining the properties of different realities. If you are interested about it's development, you can read all about it at the dev-log here. It is available to play within the browser on itch.io.


Bindin' (2021)

Bindin' is a game made for the GMTK Game Jam 2021 based on the theme "Joined Together". It was made in collaboration with a friend who made all the art. It's a puzzle game where you have the ability to bind different control inputs (i.e. WASD) to various objects in order to interact with them and solve puzzles. If you are interested about it's development, you can read all about it at the dev-log here. It is available to play within the browser on itch.io.
